# .github

Organization-wide GitHub configuration and community health files for [inators](https://github.com/inators).

## Contents

- **`PULL_REQUEST_TEMPLATE.md`** — Default PR template for all repos
- **`ISSUE_TEMPLATE/`** — Default issue forms (bug report, feature request)
- **`CODE_OF_CONDUCT.md`** — Contributor Covenant
- **`CONTRIBUTING.md`** — Contribution guidelines
- **`SECURITY.md`** — Vulnerability reporting policy
- **`SUPPORT.md`** — Where to get help
- **`profile/README.md`** — Org profile displayed on github.com/inators

## New Repos

Create new repositories from the [`template`](https://github.com/inators/template) repo — includes dependabot, CODEOWNERS, CI workflows, and a setup checklist.
