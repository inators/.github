# Support

## Getting Help

- **Bug reports** — Open an issue using the bug report template on the relevant repository
- **Feature requests** — Open an issue using the feature request template, or start a discussion in [Ideas](https://github.com/orgs/inators/discussions/categories/ideas)
- **Questions** — Ask in [Q&A Discussions](https://github.com/orgs/inators/discussions/categories/q-a)
- **General discussion** — Join the [community discussions](https://github.com/orgs/inators/discussions)

## What Not to Use Issues For

- General questions (use Discussions instead)
- Security vulnerabilities (see [SECURITY.md](SECURITY.md))
